Frequently Asked Questions

What exactly is a "Landing Zone"?

An AWS Landing Zone is a pre-configured, secure AWS environment that includes multi-account structure, identity management, logging, security services, and compliance guardrails. It's the foundation you need before deploying production applications.

How long does deployment really take?

Initial deployment takes 48 hours from contract signature. This includes AWS account setup, infrastructure provisioning, and security configuration. We handle everything - you just provide your AWS root account credentials.

Do I need an existing AWS account?

No! We can create a new AWS Organization for you, or work with your existing AWS account. Either way works.

What happens after the pilot period?

Your 50% discount locks in for life as long as you remain a customer. After 30 days, you can cancel anytime with 30 days notice, or continue at the discounted rate.

Is this Infrastructure-as-Code?

Yes! Everything is deployed using Terraform (Infrastructure-as-Code). You get full access to the code repository and can customize as needed. We also handle ongoing maintenance and updates.

What AWS services do you actually deploy?

AWS Organizations, IAM Identity Center, Security Hub, GuardDuty, Config, CloudTrail, VPC, RDS Aurora, DynamoDB, Lambda, S3, KMS, CloudWatch, SNS, and more. Full infrastructure manifest provided during onboarding.

Can I customize the infrastructure?

Absolutely. The base configuration covers 95% of use cases, but we offer customization for specific requirements. Custom work is quoted separately and typically ranges from $2K-$10K depending on scope.

Do you handle compliance audits?

We provide the compliant infrastructure and documentation needed for SOC 2, HIPAA, or FedRAMP audits. You'll still need to work with an auditor for certification, but we give you 80% of what you need.
